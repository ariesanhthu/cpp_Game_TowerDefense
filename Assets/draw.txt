0. BitBlt (Bit Block Transfer)
Hàm này dùng để sao chép một phần của bitmap từ một vùng vào một vùng khác.
Cú pháp:
BOOL BitBlt(
    HDC hdcDest,   // HDC đích (nơi vẽ)
    int nXDest,    // Toạ độ X đích
    int nYDest,    // Toạ độ Y đích
    int nWidth,    // Chiều rộng của bitmap
    int nHeight,   // Chiều cao của bitmap
    HDC hdcSrc,    // HDC nguồn (bitmap cần vẽ)
    int nXSrc,     // Toạ độ X trong bitmap nguồn
    int nYSrc,     // Toạ độ Y trong bitmap nguồn
    DWORD dwRop    // Mã thao tác (ROP: Raster Operation Code)
);
Mô tả: BitBlt sao chép một phần hoặc toàn bộ bitmap từ nguồn sang đích theo các tham số đã chỉ định. Đây là một trong những hàm mạnh mẽ nhất để vẽ bitmap.

1. StretchBlt
Mô tả: Sao chép bitmap từ nguồn (HDC) sang đích (HDC) nhưng cho phép thay đổi kích thước bitmap (phóng to hoặc thu nhỏ).
Cú pháp:
BOOL StretchBlt(
    HDC hdcDest,      // Ngữ cảnh thiết bị đích
    int xDest,        // Vị trí X trên HDC đích
    int yDest,        // Vị trí Y trên HDC đích
    int wDest,        // Chiều rộng đích
    int hDest,        // Chiều cao đích
    HDC hdcSrc,       // Ngữ cảnh thiết bị nguồn
    int xSrc,         // Vị trí X nguồn
    int ySrc,         // Vị trí Y nguồn
    int wSrc,         // Chiều rộng nguồn
    int hSrc,         // Chiều cao nguồn
    DWORD rop         // Raster operation (VD: SRCCOPY)
);
Ứng dụng: Khi bạn cần thay đổi kích thước bitmap trong quá trình sao chép.

2. TransparentBlt
Mô tả: Sao chép bitmap từ nguồn sang đích, nhưng bỏ qua các pixel có màu nhất định (transparent color).
Cú pháp:
BOOL TransparentBlt(
    HDC hdcDest,      // Ngữ cảnh thiết bị đích
    int xDest,        // Vị trí X trên HDC đích
    int yDest,        // Vị trí Y trên HDC đích
    int wDest,        // Chiều rộng đích
    int hDest,        // Chiều cao đích
    HDC hdcSrc,       // Ngữ cảnh thiết bị nguồn
    int xSrc,         // Vị trí X nguồn
    int ySrc,         // Vị trí Y nguồn
    int wSrc,         // Chiều rộng nguồn
    int hSrc,         // Chiều cao nguồn
    UINT crTransparent // Màu transparent
);
Ứng dụng: Dùng để sao chép bitmap mà không vẽ các pixel có màu cụ thể (ví dụ: màu nền).

3. AlphaBlend
Mô tả: Sao chép bitmap từ nguồn sang đích với hỗ trợ alpha blending (độ trong suốt).
Cú pháp:
BOOL AlphaBlend(
    HDC hdcDest,        // Ngữ cảnh thiết bị đích
    int xDest,          // Vị trí X trên HDC đích
    int yDest,          // Vị trí Y trên HDC đích
    int wDest,          // Chiều rộng đích
    int hDest,          // Chiều cao đích
    HDC hdcSrc,         // Ngữ cảnh thiết bị nguồn
    int xSrc,           // Vị trí X nguồn
    int ySrc,           // Vị trí Y nguồn
    int wSrc,           // Chiều rộng nguồn
    int hSrc,           // Chiều cao nguồn
    BLENDFUNCTION blend // Cấu trúc kiểm soát alpha blending
);

BLENDFUNCTION chứa thông tin về kiểu blending (tỷ lệ alpha cho từng pixel, v.v.).
Ứng dụng: Dùng khi cần vẽ bitmap với hiệu ứng trong suốt hoặc mờ dần.

4. MaskBlt
Mô tả: Sao chép bitmap từ nguồn sang đích với hỗ trợ mặt nạ (mask) để xác định các vùng hiển thị.
Cú pháp:
BOOL MaskBlt(
    HDC hdcDest,      // Ngữ cảnh thiết bị đích
    int xDest,        // Vị trí X trên HDC đích
    int yDest,        // Vị trí Y trên HDC đích
    int wDest,        // Chiều rộng bitmap
    int hDest,        // Chiều cao bitmap
    HDC hdcSrc,       // Ngữ cảnh thiết bị nguồn
    int xSrc,         // Vị trí X nguồn
    int ySrc,         // Vị trí Y nguồn
    HBITMAP hbmMask,  // Bitmap mặt nạ
    int xMask,        // Vị trí X của mặt nạ
    int yMask,        // Vị trí Y của mặt nạ
    DWORD rop         // Raster operation
);
Ứng dụng: Khi cần sao chép bitmap mà chỉ vẽ các vùng được xác định bởi mặt nạ.

5. PlgBlt
Mô tả: Sao chép bitmap từ nguồn sang đích với biến dạng hình học (parallelogram transformation).
Copy code
BOOL PlgBlt(
    HDC hdcDest,           // Ngữ cảnh thiết bị đích
    const POINT* lpPoint,  // Mảng 3 điểm xác định parallelogram
    HDC hdcSrc,            // Ngữ cảnh thiết bị nguồn
    int xSrc,              // Vị trí X nguồn
    int ySrc,              // Vị trí Y nguồn
    int wSrc,              // Chiều rộng nguồn
    int hSrc,              // Chiều cao nguồn
    HBITMAP hbmMask,       // Bitmap mặt nạ (tùy chọn)
    int xMask,             // Vị trí X của mặt nạ
    int yMask              // Vị trí Y của mặt nạ
);
Ứng dụng: Khi cần sao chép bitmap với hiệu ứng biến dạng hoặc phối cảnh.

WS_CHILD (0x40000000L): Kiểu cửa sổ con, phải có một cửa sổ cha (parent window).

WS_VISIBLE (0x10000000L): Kiểu cửa sổ được hiển thị ngay khi được tạo.

WS_CAPTION (0x00C00000L): Kiểu cửa sổ có thanh tiêu đề (title bar), bao gồm WS_BORDER và WS_DLGFRAME.
WS_BORDER (0x00800000L): Kiểu cửa sổ có khung viền.
WS_DLGFRAME (0x00400000L): Kiểu cửa sổ có khung viền kiểu hộp thoại nhưng không có thanh tiêu đề.
WS_VSCROLL (0x00200000L): Kiểu cửa sổ có thanh cuộn dọc.
WS_HSCROLL (0x00100000L): Kiểu cửa sổ có thanh cuộn ngang.
WS_SYSMENU (0x00080000L): Kiểu cửa sổ có menu hệ thống (system menu) trên thanh tiêu đề.
WS_THICKFRAME (0x00040000L): Kiểu cửa sổ có khung viền có thể thay đổi kích thước.
WS_GROUP (0x00020000L): Kiểu cửa sổ nhóm, thường được sử dụng trong các hộp thoại để nhóm các điều khiển.
WS_TABSTOP (0x00010000L): Kiểu cửa sổ có thể nhận được tiêu điểm khi người dùng nhấn phím Tab.
WS_MINIMIZEBOX (0x00020000L): Kiểu cửa sổ có nút thu nhỏ.
WS_MAXIMIZEBOX (0x00010000L): Kiểu cửa sổ có nút phóng to.
